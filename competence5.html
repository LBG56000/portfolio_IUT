Site de streaming Scooby-Doo

Compétence: Conduite de projet

Recueil des besoins du client

Pour constituer le site web demandé nous avons dans un premier déterminer ce que le client voulait voir apparaître sur son site de streaming vidéo. Nous avons donc rédigé un rapport avec une liste des fonctionnalités techniques.

Charte graphique

La charte graphique du site correspondait à la réalisation d’un document ou nous avons rassemblé toutes les informations comme logo, taille de polices, polices et couleurs que nous allions utiliser.

Maquettage

Le maquettage consistait à montrer une première ébauche de notre site web avec les différentes couleurs, polices. Lors de cette maquette nous avons pu opérer certain ajustement sur des positionnement, alignement ou encore voir les différents point de la responsivité que nous devions prévoir. Les ajustements ont pu se faire via une communication directe avec le client pour lui livrer un site qui lui ressemblait.

Codage

Cette partie constitue l'aboutissement de notre travail sur l’ensemble du projet, en effet grâce aux recueil du besoin du client, la charte graphique ou encore la maquette nous avons pu coder un site en HTML, CSS et quelques éléments en JavaScript pour rendre notre site utilisable pour le client.
