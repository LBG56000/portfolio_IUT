Installation d'un poste pour le développement

Ce projet consistait à organiser l'automatisation de certaines taches à accomplir

Ce projet consistait en la récupération d'information depuis des programmes en langage C préalablement créé.
Le projet était divisé en deux parties

  1. La partie conceptualisation
  2. La partie automatisation

La conceptualisation

Cette partie était de prendre connaissance des différents rendus que nous devions remmetre à la fin du projet.
Ainsi nous avons pu faire un liste des différentes taches à faire pour cette première partie

  • Création de trois programmes en langage C
  • Création d'une documentation utiisateur.ice/technique des programmes

Les programmes en C

Les créations devaient se conforter aux éxigences du client

Ils devaient respester plusieurs conditions comme le fait d'être composé de strutures

Lien vers les fichiers C

Les fichiers HTLM

Via les fichiers C nous devaons dans un premier temps faire la documentation utilisateur à la main via les commenatire que nous avions récupéré

Voir la documentation technique HTML

La documentation utilisateur devait expliquer notre mode de production sur les différents rendus du projet

L'automatisation

La partie principale du projet consistait à automatiser la génération des deux documentation via le langage PHP

PHP

PHP est reconnu pour un langage près du développement web donc parfait pour transformer un fichier en C en HTML

En effet dans nos commentaires nous devions respecter une certainer nomenclature

montrer les différentes façon de commenter

Le langage PHP nous a servit pour permettre de parcourir un fichier en langage C ou Markdown pour permettre de produire deux fichiers HTML (documentation technique et utiisateur)

Nous avons pu ainsi à travers les fichiers HTML les transformer en PDF comme convenu avec le client avec une image Docker utilisé pour l'occasion

Utilisation de Docker

La conteneurisation est utilisé pour permettre l'automatisation plus facile avec Docker.

Ainsi nous avons eu accès à deux images (programme) permettant de faire de nos fichier html des fichiers pdf.
